Admin Panel - Full Stack Web Application

A full-stack Admin Dashboard application built using React (Frontend) and Node.js with Express (Backend). This project demonstrates authentication, dashboard UI, CRUD operations, and responsive design.

Features

User Authentication (Login System,JWT Authentication,
Role-based access control)
Responsive Admin Dashboard
Sidebar and Navbar Navigation
Dashboard Statistics Cards
CRUD Operations (Create, Read, Update, Delete)
Settings Page with Toggle Options
Responsive design for mobile, tablet, and desktop
REST API integration between frontend and backend

Tech Stack

Frontend:
React.js
HTML5
CSS3
JavaScript (ES6+)

Backend:
Node.js
Express.js

Database:
MongoDB

Installation & Setup

1. Clone Repository
git clone https://github.com/your-username/admin-panel.git
cd admin-panel

2. Frontend Setup
cd admin-frontend
npm install
npm run dev

Frontend runs on:
http://localhost:5173

3. Backend Setup
cd admin-backend
npm install
node server.js

Backend runs on:
http://localhost:5000

API Integration

Frontend communicates with backend using REST APIs for authentication, data fetching, and CRUD operations.

Responsive Design

Works on desktop, tablet, and mobile devices.
